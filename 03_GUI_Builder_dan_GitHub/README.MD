# GUI Builder dan GitHub

## Deskripsi
Folder ini berisi materi praktikum tentang penggunaan GUI Builder untuk membuat antarmuka pengguna dan pengelolaan proyek menggunakan GitHub. Mahasiswa akan belajar cara membuat aplikasi desktop dengan antarmuka grafis dan mengelola kode sumber menggunakan version control.

## Materi yang Dibahas
1. GUI Builder di Visual Studio
   - Windows Forms Designer
   - WPF Designer (XAML)
   - Komponen-komponen GUI
   - Event handling
   - Layout management

2. GitHub dan Version Control
   - Konsep dasar Git
   - Repository management
   - Branching dan merging
   - Collaboration workflow
   - Pull requests dan code review
   - Issue tracking

3. Best Practices
   - Responsive design
   - User experience (UX) principles
   - Git commit conventions
   - Documentation standards
   - Project organization
