# Clean Code

## Deskripsi
Folder ini berisi materi praktikum tentang prinsip dan praktik Clean Code dalam pengembangan perangkat lunak. Mahasiswa akan mempelajari bagaimana menulis kode yang bersih, mudah dibaca, dan mudah dipelihara sesuai dengan prinsip-prinsip yang diakui industri.

## Materi yang Dibahas
1. Clean Code Principles
   - Meaningful names
   - Functions
   - Comments
   - Formatting
   - Objects and data structures
   - Error handling
   - Boundaries
   - Unit tests

2. Code Smells dan Refactoring
   - Identifying code smells
   - Refactoring techniques
   - Code metrics
   - Technical debt
   - Legacy code handling

3. SOLID Principles
   - Single Responsibility Principle
   - Open/Closed Principle
   - Liskov Substitution Principle
   - Interface Segregation Principle
   - Dependency Inversion Principle

4. Clean Architecture
   - Separation of concerns
   - Dependency management
   - Layer separation
   - Interface design
   - Package organization
