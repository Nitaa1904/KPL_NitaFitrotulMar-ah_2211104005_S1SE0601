# Generics dalam C#

## Deskripsi
Folder ini berisi materi praktikum tentang penggunaan Generics dalam C#. Mahasiswa akan mempelajari konsep dan implementasi generic programming untuk membuat kode yang lebih fleksibel, type-safe, dan reusable.

   - Konsep dasar generic programming
   - Type parameters
   - Generic constraints
   - Type inference
   - Benefits of using generics

2. Generic Collections

   - Dictionary<TKey, TValue>
   - Queue<T>
   - Stack<T>
   - Custom generic collections

3. Generic Methods dan Classes
   - Generic method implementation
   - Generic class design
   - Generic interfaces
   - Covariance dan contravariance
   - Generic constraints advanced

4. Best Practices
   - When to use generics
   - Performance considerations
   - Common patterns
   - Error handling
   - Documentation
