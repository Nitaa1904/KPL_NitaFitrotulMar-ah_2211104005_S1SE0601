# Pengenalan IDE dan Pemrograman C#

## Deskripsi
Folder ini berisi materi praktikum tentang pengenalan Integrated Development Environment (IDE) dan dasar-dasar pemrograman C#. Mahasiswa akan belajar menggunakan Visual Studio sebagai IDE utama dan memahami konsep dasar bahasa pemrograman C#.

## Materi yang Dibahas
1. Pengenalan Visual Studio IDE
   - Interface dan fitur utama
   - Debugging tools
   - Solution dan project management
   - Code editor dan IntelliSense

2. Dasar-dasar C#
   - Sintaks dasar C#
   - Tipe data dan variabel
   - Control flow (if, switch, loop)
   - Method dan Class
   - Object-Oriented Programming basics

3. Praktik
   - Membuat project C# pertama
   - Debugging sederhana
   - Best practices dalam penulisan kode
   - Konvensi penamaan di C#
