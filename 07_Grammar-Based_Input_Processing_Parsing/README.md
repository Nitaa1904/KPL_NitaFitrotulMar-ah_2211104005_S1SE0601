# Grammar-Based Input Processing dan Parsing

## Deskripsi
Folder ini berisi materi praktikum tentang pemrosesan input berbasis grammar dan teknik parsing. Mahasiswa akan mempelajari bagaimana menganalisis dan memproses input terstruktur menggunakan konsep formal grammar dan implementasi parser.

## Materi yang Dibahas
1. Dasar-dasar Grammar
   - Context-Free Grammar (CFG)
   - Regular Expressions
   - BNF dan EBNF notation
   - Syntax diagrams
   - Grammar rules dan productions

2. Teknik Parsing
   - Recursive descent parsing
   - Top-down parsing
   - Bottom-up parsing
   - Parser generators
   - Abstract Syntax Trees (AST)

3. Implementasi dalam C#
   - Lexical analysis
   - Token generation
   - Parser implementation
   - AST construction
   - Semantic analysis
   - Error handling dan recovery
