# Design by Contract dan Defensive Programming

## Deskripsi
Folder ini berisi materi praktikum tentang pendekatan Design by Contract (DbC) dan Defensive Programming dalam pengembangan perangkat lunak. Mahasiswa akan mempelajari bagaimana membuat kode yang lebih robust dan reliable dengan menerapkan kontrak dan teknik defensive programming.

## Materi yang Dibahas
1. Design by Contract (DbC)
   - Preconditions
   - Postconditions
   - Invariants
   - Contract programming dalam C#
   - Code Contracts library

2. Defensive Programming
   - Input validation
   - Error handling
   - Exception management
   - Null checking
   - Boundary testing
   - Assertions dan debugging

3. Implementasi dalam C#
   - Contract annotations
   - Debug.Assert statements
   - Exception handling patterns
   - Parameter validation
   - Secure coding practices
