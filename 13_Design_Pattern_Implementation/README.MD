# Design Pattern Implementation

## Deskripsi
Folder ini berisi materi praktikum tentang implementasi design patterns dalam pengembangan perangkat lunak. Mahasiswa akan mempelajari berbagai pola desain dan bagaimana menerapkannya dalam konteks yang tepat untuk membuat kode yang lebih maintainable dan flexible.

## Materi yang Dibahas
1. Creational Patterns
   - Singleton
   - Factory Method
   - Abstract Factory
   - Builder
   - Prototype
   - Dependency Injection

2. Structural Patterns
   - Adapter
   - Bridge
   - Composite
   - Decorator
   - Facade
   - Proxy
   - Flyweight

3. Behavioral Patterns
   - Observer
   - Strategy
   - Command
   - State
   - Template Method
   - Iterator
   - Mediator

4. Enterprise Patterns
   - Repository
   - Unit of Work
   - Service Locator
   - Specification
   - Event Sourcing
   - CQRS

## Tujuan Pembelajaran
1. Memahami berbagai design patterns
2. Mengenali kapan menggunakan pattern tertentu
3. Mampu mengimplementasikan patterns dengan benar
4. Mengkombinasikan multiple patterns
5. Mengevaluasi trade-offs tiap pattern
