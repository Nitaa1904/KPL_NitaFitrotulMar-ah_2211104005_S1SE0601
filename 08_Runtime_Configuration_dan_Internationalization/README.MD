# Runtime Configuration dan Internationalization

## Deskripsi
Folder ini berisi materi praktikum tentang implementasi runtime configuration dan internationalization (i18n) dalam pengembangan aplikasi. Mahasiswa akan mempelajari bagaimana membuat aplikasi yang dapat dikonfigurasi secara dinamis dan mendukung berbagai bahasa dan budaya.

## Materi yang Dibahas
1. Runtime Configuration
   - Configuration files
   - Environment variables
   - Command-line arguments
   - Dynamic settings management
   - Configuration providers

2. Internationalization (i18n)
   - Resource files
   - String localization
   - Culture-specific formatting
   - Right-to-left (RTL) support
   - Multi-language support

3. Implementasi dalam C#
   - IConfiguration interface
   - Resource managers
   - CultureInfo handling
   - Format providers
   - Localization services
