# Pengantar Praktikum Konstruksi Perangkat Lunak

## Deskripsi
Folder ini berisi materi dan praktikum untuk pertemuan pertama mata kuliah Praktikum Konstruksi Perangkat Lunak. Pada pertemuan ini, mahasiswa akan diperkenalkan dengan konsep dasar konstruksi perangkat lunak dan lingkungan pengembangan yang akan digunakan selama semester.

## Materi yang Dibahas
1. Pengenalan Konstruksi Perangkat Lunak
   - Definisi dan ruang lingkup
   - Pentingnya konstruksi perangkat lunak yang baik
   - Best practices dalam pengembangan software

2. Persiapan Lingkungan Pengembangan
   - Setup tools dan software yang diperlukan
   - Pengenalan version control (Git)
   - Pengenalan IDE yang akan digunakan

3. Overview Materi Semester
   - Roadmap pembelajaran
   - Sistem penilaian
   - Aturan praktikum
